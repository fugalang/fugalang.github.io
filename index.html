fugu

Системный язык программирования нового поколения

О проекте

fugu - это системный язык программирования, разработанный для создания низкоуровневого ПО: ОС, драйверов и высокопроизводительных систем.

  • Читаемый синтаксис
  • Предсказуемое поведение
  • Безопасность памяти

Компилятор

Стартовый компилятор fugus написан на Go и использует табличный парсер и оптимизации.

Быстрый старт

$ git clone https://github.com/fugalang/fugu
$ cd fugu
$ make install
