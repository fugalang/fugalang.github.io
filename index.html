fugu

Системный язык программирования нового поколения

О проекте

fugu - это системный язык программирования, разработанный специально для создания низкоуровневого ПО, такого как операционные системы и драйверы. Проект ориентирован на максимальную производительность и прямой контроль над аппаратными ресурсами.

Ключевые принципы разработки:

  • Читаемый синтаксис - код должен быть понятным и легко поддерживаемым.
  • Предсказуемость - отсутствие скрытого поведения компилятора и рантайма.
  • Безопасность памяти - строгий контроль без ущерба для скорости.

Компилятор fugu

Быстрый и легковесный компилятор, написанный на Go. Использует максимально эффективные механизмы разбора исходного кода (парсинга) и генерации целевого бинарного кода.

Быстрый старт

Соберите компилятор из исходников прямо сейчас:

$ git clone https://github.com
$ cd fugu
$ make install
